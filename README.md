
# Employee Satisfaction and Company Performance Analysis

## Overview

This project analyzes employee reviews, company ratings, workplace sentiments, and hiring trends using data collected from AmbitionBox. The objective is to transform raw employee feedback into actionable business insights through data cleaning, exploratory data analysis, sentiment analysis, SQL querying, and interactive Power BI dashboards.

The project demonstrates an end-to-end Data Analytics workflow using Python, MySQL, and Power BI.

## Project Objectives

* Analyze employee satisfaction across companies.
* Identify factors influencing company ratings.
* Understand hiring and interview trends.
* Perform sentiment analysis on employee feedback.
* Build interactive dashboards for business decision-making.
* Generate actionable insights from employee review data.

## Key Insights

* Companies with higher ratings generally receive more positive employee feedback.
* Learning opportunities and workplace culture are frequently mentioned positive factors.
* Salary concerns and workload pressure are common employee complaints.
* Companies with larger review volumes tend to have higher hiring activity.
* Company performance can be effectively categorized into Excellent, Good, and Poor groups based on ratings.
